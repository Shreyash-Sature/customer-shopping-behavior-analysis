# 🛍️ Customer Shopping Behavior Analysis

> An end-to-end retail data analytics project that analyzes customer shopping behavior using **Python, PostgreSQL, and Power BI**. The project focuses on extracting actionable business insights through data cleaning, exploratory data analysis (EDA), SQL analytics, and interactive dashboards.

---

## 📌 Project Overview

Understanding customer purchasing behavior is essential for improving business performance and customer satisfaction. This project analyzes a retail transaction dataset containing **3,900 customer purchase records** to identify purchasing trends, customer segments, product performance, and revenue insights.

The project demonstrates a complete analytics pipeline—from raw data preprocessing to business intelligence dashboard creation.

---

## 🎯 Objectives

- Clean and preprocess raw retail transaction data.
- Perform Exploratory Data Analysis (EDA).
- Engineer meaningful features for analysis.
- Store processed data in PostgreSQL.
- Answer real-world business questions using SQL.
- Build an interactive Power BI dashboard.
- Generate data-driven business recommendations.

# 📌 Key Insights

- Clothing category generated the highest revenue.
- Young adults contributed the largest share of total revenue.
- Express shipping customers spent slightly more on average.
- Loyal customers represented the largest customer segment.
- Certain products had significantly higher discount dependency.
- Subscribers and non-subsscribers exhibited different purchasing patterns.
